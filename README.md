# PA ATV Map - Fixed Bottom Toolbar

Upload all files in this folder to the root of your GitHub repository: `PA_ATV_map_app`.

Open:
https://melonz1-beep.github.io/PA_ATV_map_app/?v=400

If Chrome shows an old version, clear site data for the GitHub Pages URL and open the link again.

This version includes a fixed two-row bottom toolbar, pinch zoom, pan, double-tap zoom, GPS blue dot, trail buttons, marker filters, favorite save, Google Maps navigation for GPS location, and offline support.

GPS placement is approximate because it is based on an image map. Always follow posted DCNR signs and official rules.
