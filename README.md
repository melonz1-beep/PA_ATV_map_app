# PA ATV Map - GitHub Pages App

This is a ready-to-upload offline Progressive Web App for the Northcentral Regional ATV Trail map.

## Upload to GitHub Pages

1. Create a new GitHub repository named `pa-atv-map`.
2. Upload every file in this folder to the repository root.
3. Go to **Settings > Pages**.
4. Under **Build and deployment**, choose **Deploy from a branch**.
5. Select **main** and **/(root)**, then save.
6. Open: `https://melonz1-beep.github.io/pa-atv-map/`
7. On Android/Chrome, tap the three dots and choose **Add to Home screen** or **Install app**.

## Offline use

Open the app once while you have service or Wi-Fi. The service worker stores the map, icons, and app files so it opens offline later.

## GPS note

The blue location dot uses the phone's GPS and an approximate map calibration. It is intended for general trail awareness only. Always follow posted DCNR trail signs and official rules.

## QR code

`install-qr.png` points to `https://melonz1-beep.github.io/pa-atv-map/`. If you use a different GitHub username or repository name, make a new QR code for your new link.
