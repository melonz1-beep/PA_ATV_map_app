PA ATV GPS Trail Map App
Upload these files to the root of your GitHub repository: PA_ATV_map_app.
Open: https://melonz1-beep.github.io/PA_ATV_map_app/?v=200
Features:
Full offline map
Pinch zoom and pan
Trail section buttons
GPS blue dot using phone GPS
Follow-me mode
Gas, parking, and camping markers with filter buttons
GPS note: The blue dot is calibrated to the printed coordinate grid on the DCNR map. It is intended for trail awareness, not as a legal or safety navigation source. Always follow DCNR signs and posted routes.
