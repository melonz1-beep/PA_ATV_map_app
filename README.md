# PA ATV GPS Trail App - Pro Upgrade

Upload these files to the root of your existing `PA_ATV_map_app` GitHub repository.

Open after upload:
https://melonz1-beep.github.io/PA_ATV_map_app/?v=300

Features:
- Offline zoomable map
- Pinch zoom, drag/pan, and double-tap zoom
- GPS blue dot and follow-me mode
- Compass heading when supported by the phone/browser
- Trailhead distance list
- Favorites saved on your phone
- Google Maps navigation buttons
- Trail info, camping info, emergency reminders, DCNR/NRAT rules
- Gas, parking, camping, food marker filters

GPS and marker locations are approximate because this is an image-based map using approximate calibration from the map coordinate grid. Always follow posted DCNR trail signs and official rules.
